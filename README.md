## 🚀 How to Set Up and Run the Code Locally

### 1. Clone the playlist repository
```bash
git clone https://github.com/msumar7426/Python_Course.git
cd Python_Course
```

### 2. Set up a virtual environment
Set up a clean virtual environment to keep dependencies isolated:
```bash
python3 -m venv python_course_venv
source python_course_venv/bin/activate
```

### 3. Run a script
Choose any Python script to execute (e.g., the Text Analyzer):
```bash
python3 01_functions_variables/mini_project/Text_Analyzer.py
```
