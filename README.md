# Harvard CS50P: Programming Foundations for ML, NLP & Data Science

Welcome to my Python learning repository! This project tracks my journey through **Harvard University's CS50 Introduction to Programming with Python (CS50P)**.

Rather than just a basic coding challenge, this repository is designed as a professional-grade portfolio. It documents my progression from core programming concepts to advanced paradigms (like Regular Expressions and Object-Oriented Programming), laying a rock-solid software engineering foundation necessary for **Machine Learning (ML)**, **Natural Language Processing (NLP)**, and **Data Science**.

## 📈 Path to ML, NLP & Data Science Readiness
Each section in this course builds critical skills directly applicable to advanced analytics fields:
* **Text Operations & Regex (01, 08):** Essential for **NLP (Natural Language Processing)** preprocessing (tokenization, text sanitization, and data parsing).
* **Data Processing & File I/O (03, 07):** Prepares for loading, filtering, and converting raw datasets into structured formats.
* **OOP & Testing (06, 09):** Essential for building scalable production-grade model training pipelines, custom estimators, and robust software architectures.

## 🛠️ How to Set Up and Run locally

### 1. Clone the repository
```bash
git clone https://github.com/msumar7426/Python_Course.git
cd Python_Course
```

### 2. Set up virtual environment
```bash
python3 -m venv python_course_venv
source python_course_venv/bin/activate
```

### 3. Run a sample script
Choose any script to execute (e.g., the Text Analyzer mini-project):
```bash
python3 01_functions_variables/mini_project/Text_Analyzer.py
```
